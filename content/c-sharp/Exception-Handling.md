#### 1. What is an exception?

- It is an event that happens during the execution of a program that disrupts the flow of the program or crashes it.

#### 2. What is exception handling?

- It is a mechanism in programming that deals with exceptions in a controlled manner.
  reckoned

## Exception handling syntax

#### Keywords

1. try
2. catch
3. throws
4. finally

#### 3. What is the purpose of the try block?

- The try block contains code that can throw/cause exceptions.

#### 4. What is the purpose of the catch block?

- The catch block specifies the exception type and contains a set of instructions to be executed when the exception happens.

#### 5. Can a try block multiple catch blocks? if so, what is it's purpose?

- Multiple catch blocks with a single try block is used to handle different types of exceptions.

- Each catch block is designed to handle a specific type of exception.
